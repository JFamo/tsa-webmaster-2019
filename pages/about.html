About This Website

This website was built by team 2004-901 and submitted to the Pennsylvania Technology Student Association Region 9 conference.

It was built entirely using custom HTML, Javascript, and CSS, with use of the open source Bootstrap library, found here : https://getbootstrap.com/ and the open source Smoothscroll JS library, found here : SmoothScrollJS.

No templates or content frameworks were used in the creation of this website. It is hosted on Github via Github Pages.
